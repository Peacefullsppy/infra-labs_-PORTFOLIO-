# Infra Labs — Portfólio de Infraestrutura de TI

Bem-vindo ao meu repositório de portfólio prático em **Infraestrutura de TI**.

Este projeto reúne laboratórios, configurações, testes e documentações desenvolvidos durante meus estudos nas áreas de **suporte técnico, redes, sistemas operacionais, servidores e monitoramento**. O objetivo é demonstrar, de forma organizada, minha evolução técnica e minha capacidade de aplicar conhecimentos em ambientes simulados.

---

## Sobre o portfólio

Os laboratórios deste repositório foram criados para desenvolver experiência prática com atividades comuns da área de infraestrutura, como:

- Criação e configuração de máquinas virtuais;
- Instalação e administração de servidores Linux;
- Utilização de comandos de diagnóstico;
- Configuração e verificação de redes;
- Gerenciamento de usuários, arquivos e permissões;
- Instalação e configuração de serviços;
- Monitoramento de recursos computacionais;
- Identificação e resolução de problemas;
- Produção de documentação técnica.

Cada atividade possui uma documentação própria, contendo o objetivo, as configurações utilizadas, os procedimentos realizados, os resultados, as dificuldades encontradas e as evidências do laboratório.

## Laboratórios disponíveis

### Laboratório 01 — Instalação e validação de um servidor Ubuntu

Criação de uma máquina virtual no Oracle VirtualBox, instalação do Ubuntu Server, realização do primeiro login e execução de comandos básicos para validar o ambiente.

**Principais comandos utilizados:**

```bash
hostname
whoami
ip a
df -h
free -h
```

**Conhecimentos demonstrados:**

- Criação de máquinas virtuais;
- Instalação de sistema operacional para servidor;
- Acesso ao terminal Linux;
- Identificação do hostname e do usuário autenticado;
- Consulta das interfaces e dos endereços de rede;
- Verificação do espaço em disco;
- Análise do uso de memória RAM.

> Consulte a documentação e as evidências do laboratório na pasta correspondente 'print'.
